### 🤝 参与开发

```
git clone https://github.com/HelKyle/focus.git

cd focus

yarn && yarn dev
```

### 下载
可以在 [👉release👈](https://github.com/HelKyle/focus/releases) 页面看到发布版本，点击下载即可。
